# Mini CBT Engine

The Mini CBT Engine is an interactive web-based assessment tool built with Python and Flask. It transitions basic console-based quiz applications into a user-friendly browser interface.

## Core Features
- **Interactive Quiz Interface:** Answer a series of questions.
- **Scoring Engine:** Automatically scores answers using Object-Oriented Programming principles.
- **Review History:** Implements a Stack (LIFO) data structure allowing users to view their most recent attempts.
- **Timestamped Results:** Accurate completion times dynamically updated upon submission.

## Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS

## Beginner-Friendly Instructions to Run

1. **Clone the Repository (or navigate to the folder):**
   ```bash
   cd mini_cbt
   ```

2. **Install Flask:**
   Make sure you have Python installed. It is recommended to use a virtual environment.
   ```bash
   pip install -r requirements.txt
   ```
   Or simply:
   ```bash
   pip install Flask==3.0.0
   ```

3. **Run the Application:**
   ```bash
   python app.py
   ```

4. **Access the Web App:**
   Open your browser and navigate to exactly what the terminal says, usually:
   `http://127.0.0.0:5000` or `http://localhost:5000`

Enjoy testing your knowledge!
